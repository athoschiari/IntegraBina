Instalação do pacote

Inicialmente você deve instalar o pacote de desenvolvimento "PlugInRepeater-1-XP-Vista-Win7.exe" em sua máquina de desenvolvimento. 




LEMBRE-SE:
- Você deve instalar o Pacote do PlugIn em cada cliente seu. 
- Este pacote não instala os exemplos e nem os fontes em seu cliente 
- No seu aplicativo você deve informar a pasta e o nome do PlugIn, veja nos aplicativos-exemplos como fazer isto.

- Não há necessidade de registar nenhuma ocx manualmente. Tudo será automático.
- Existem várias formas de instalação, escolha uma abaixo

- Para instalar o PlugIn em seu cliente e registrar as bibliotecas, proceda como abaixo:
-------------------------------------------------------------------------------------------------------------------------


Formas de instalação no cliente:
-------------------------------------------------------------------------------------------------------------------------
PlugInRepeater-1-XP-Vista-Win7.exe
- Mostrará todas as telas e o usuário poderá escolher uma pasta
- Se a pasta não existir será criada
-------------------------------------------------------------------------------------------------------------------------

-------------------------------------------------------------------------------------------------------------------------
PlugInRepeater-1-XP-Vista-Win7.exe /VERYSILENT
- NÃO Mostrará nenhuma tela e instalará na pasta padrão "\PlugInRepeater"
- Se a pasta não existir será criada
-------------------------------------------------------------------------------------------------------------------------

-------------------------------------------------------------------------------------------------------------------------
PlugInRepeater-1-XP-Vista-Win7.exe /SILENT
- Mostrará a tela de progresso e instalará na pasta padrão "\PlugInRepeater"
- Se a pasta não existir será criada
-------------------------------------------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------------------------------------------
PlugInRepeater-1-XP-Vista-Win7.exe /DIR="C:\Pasta" /VERYSILENT 
- Não mostrará nenhuma tela e instalará na pasta indicada em /DIR="C:\Pasta"
- (O nome da pasta deverá ser entre aspas)
- Se a pasta não existir será criada
-------------------------------------------------------------------------------------------------------------------------
-------------------------------------------------------------------------------------------------------------------------
PlugInRepeater-1-XP-Vista-Win7.exe /DIR="C:\Pasta" /SILENT
- Mostrará só a tela de progresso e instalará na pasta indicada em /DIR="C:\Pasta"
- (O nome da pasta deverá ser entre aspas)
- Se a pasta não existir será criada
-------------------------------------------------------------------------------------------------------------------------
